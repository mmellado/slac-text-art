Slack Text Art

This is a Slack bot that lets you use text art using your Slack emojis. you can call it by doing /textart [text string] [symbol]. Just make sure to add the space emoji to your slack channel and install using the button on the top right. You can also use the form below to test what the bot would provide as an output.
